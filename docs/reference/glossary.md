# Glossary

Terminology used in Elsai Guardrails documentation.

## A

### API Key
Authentication credential for LLM providers.

### Async
Asynchronous operations that don't block execution.

## B

### Block
Prevent content from being processed or returned.

## C

### Configuration
Settings that control guardrail behavior.

### Confidence
Probability score indicating detection certainty.

## D

### Detection
Process of identifying problematic content.

## G

### Guardrail
Safety mechanism that validates content.

### GuardrailResult
Object containing results of guardrail checks.

### GuardrailSystem
Core class for performing guardrail checks.

## I

### Input Rails
Validation performed on user input.

### Input Validation
Process of checking user input before processing.

## L

### LLM
Large Language Model.

### LLMRails
High-level class combining LLM and guardrails.

## O

### Output Rails
Validation performed on LLM output.

### Output Validation
Process of checking LLM output before returning.

## R

### Rails
Term for guardrails/safety mechanisms.

### RailsConfig
Configuration container for the rails system.

## S

### Content Classification
Detection of content based on meaning/intent.

### Sensitive Data
Personal information that should be protected.

## T

### Threshold
Value that determines when content is blocked.

### Toxicity
Harmful, offensive, or inappropriate content.

## V

### Validation
Process of checking content against rules.
